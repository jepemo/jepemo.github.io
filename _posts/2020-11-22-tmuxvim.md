# Vim
## Move
```
h: Left
j: Down
k: Up
l: Right

w: Next word
b: Past word
e: Next word (end position)

%: Goto begin/end of ([{}])
0: Goto begin line
$: Goto end line
gg: Start file
G: End file
<num>G: Goto line <num>
```
